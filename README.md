# FanficRepo
  
Веб приложение для публикации рассказов и фанфиков  
_Мое первое приложение на спринге и вью, написанное за три недели во время мини курсов_  
## Backend:
 - REST api  
 - Security - jwt token  
 - Coudinary для хостинга картинок  
 - Hibernate

## Frontend:
 - Vue.js
 - Vuex
 - Axios
